This tool allows you to analyze the climate local to your region. After you input your location and a time range you're curious about, it will find the closest weather station and present you with historical climate data. Please note this tool is currently limited only to temperature records, and does not include precipitation or other climate information.

Please allow for up to 30 seconds for the data to load. Some stations have 130 years worth of data or more, and sometimes I must search multiple station data files to find a complete one.

All data and calculations completed using the Global Historical Climatology Network daily (GHCNd) provided by the US National Oceanic and Atmospheric Administration. This dataset is globally applicable for all land areas and usually includes station readings as recent as 3-5 days in the past. The analyses herein process this data raw; the only processing is to filter to identify weather stations with relatively complete weather records. All average temperature values are evaluated using the standard practice of taking the midpoint of the maximum and minimum temperature for every day in the record. Webpage by Tobias Schultz, 2022.
